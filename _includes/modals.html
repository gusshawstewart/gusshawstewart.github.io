Rumble in the Jungle- Boxing Game

This was the first project I did at GA. It is an 8-Bit, turn-based boxing match game. There is a power bar which is constantly moving that dictates how powerful each punch is according to the timing of the user's strike. The game was built with jQuery, animation sprites and CSS animations to create a retro, user friendly game.
